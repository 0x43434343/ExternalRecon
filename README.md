### Prerequisites


```
python3-pip
request
bs4
fake_useragent

```

### Installing


```
git clone https://github.com/faa5431/ExternalRecon.git
cd ExternalRecon
sudo apt install python3-pip
sudo pip3 install request
sudo pip3 install bs4
sudo pip3 install fake_useragent
```
